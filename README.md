## 🎯 Visão Geral

Este repositório tem como objetivo documentar meu aprendizado em Python, com foco na prática e consolidação de conceitos fundamentais, intermediários e avançados da linguagem.

O conteúdo inclui exercícios de lógica de programação, manipulação de dados, orientação a objetos, APIs REST, programação assíncrona, testes, tratamento de exceitos e muito mais.
